# Рубежный контроль №1

## Тема

Технологии разведочного анализа и обработки данных.

## Набор данных

Использован набор данных `iris` из `scikit-learn`.

Официальная документация `load_iris` указывает, что это классический набор данных для многоклассовой классификации:

- 3 класса;
- 150 объектов;
- 4 количественных признака.

Источник: https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_iris.html

## Что сделано

1. Загружен набор данных `iris`.
2. Создан искусственный категориальный признак `petal_length_category` на основе `petal length (cm)`.
3. Искусственно добавлены пропуски:
   - в количественный признак `sepal width (cm)`;
   - в категориальный признак `petal_length_category`.
4. Пропуски обработаны:
   - для количественного признака методом `median`;
   - для категориального признака методом `most_frequent`.
5. Построена гистограмма для признака `petal length (cm)`.
6. Подготовлен основной notebook `rk1.ipynb` и python-файл `report.py`.

## Какие признаки использовать дальше

Для дальнейшего построения моделей машинного обучения рекомендуется использовать исходные количественные признаки:

- `sepal length (cm)`
- `sepal width (cm)`
- `petal length (cm)`
- `petal width (cm)`

Искусственный признак `petal_length_category` лучше использовать только как вспомогательный, потому что он построен на основе `petal length (cm)` и частично дублирует его информацию.

## Запуск

```powershell
py report.py
```
