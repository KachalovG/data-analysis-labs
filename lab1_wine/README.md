# Лабораторная работа №1

## Разведочный анализ данных. Исследование и визуализация данных

В данной лабораторной работе выполнен разведочный анализ данных для набора данных **Wine** из библиотеки **Scikit-learn**.

В работе рассмотрены:
- основные характеристики датасета;
- проверка пропусков и типов данных;
- описательная статистика;
- визуальное исследование данных;
- анализ корреляции признаков.

Основной файл работы: `notebooks/lab1_wine.ipynb`
